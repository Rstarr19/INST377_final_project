# F1 Live Dashboard

## Project Description

F1 Live Dashboard is an interactive Formula 1 web application that allows users to explore live F1 driver standings, favorite drivers, and inspect detailed driver statistics through a racing themed user interface.

The web application combines live external Formula 1 APIs with a connected Supabase database and displays the data through responsive front end attributes

On this web applicaton, users can:
- View live Formula 1 driver standings
- Favorite and remove favorite drivers
- Explore driver information pages
- View live standings charts
- Interact with different F1 themed UI components

---

## Target Browsers

This application is optimized for standard desktop browsers like:

- Google Chrome
- Microsoft Edge
- Mozilla Firefox
- Safari

The application is designed primarily for desktop or laptop engagement.

---

## Front end JavaScript Libraries Used

### Chart.js
This was used for displaying live and interactive Formula 1 driver's championship standings as a bar graph

### AOS (Animated on Scroll)
This was used for animated fade in transition effects on different pages through elements like driver cards and displayed images
