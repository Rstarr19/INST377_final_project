About This Project

The F1 Dashboard is a full-stack web application that visualizes Formula 1 data, integrates live APIs, and allows users to interact with driver statistics in real time.

Live API Data

Fetches real-time Formula 1 driver standings from a backend REST API.

Data Visualization

Uses Chart.js to display driver performance and standings visually.

User Favorites

Stores favorite drivers using a Supabase database connection.

External F1 Data

Integrates third-party F1 APIs for extended driver and team information.
