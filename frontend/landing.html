F1 Live Dashboard

Welcome to the F1 Live Dashboard!

It is an interactive Formula 1 web application that tracks live driver standings, lets users select their favorite drivers, and explores current Formula 1 data.

This project uses real and updating API data, Supabase database integration, animated UI components, and interactive charts to create a complete motorsport dashboard experience.

Live Standings

Track current F1 driver positions and championship points.

Favorites System

Favorite your favorite drivers and sync them across the dashboard.

Interactive Charts

Visualize standings data using animated Chart.js graphics.
